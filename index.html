CS 145 - Databases

Course on using databases, designing scalable data systems, and SQL+Python project on BigQuery

Course Overview

This course covers how to use databases in applications, first principles on how to scale for large data sets and how to design good data systems.

  • Introduction to relational data model, relational database engines, and SQL.
  • How to scale systems for large data sets on servers and server clusters
  • How to design good schemas based on dependencies, normal forms including indexes, views, and transactions.
  • Programming project involving SQL+Python querying and analyzing terabytes of data on BigQuery
